# Meal Planner App

A simple and user-friendly web app to plan meals for the week. Users can add, edit, and delete meals for each day, and the app remembers their plans even after refreshing the page.

---

## Features
- Add meals for **Breakfast, Lunch, Dinner, or Snack**
- Select the **day of the week** for each meal
- Meals are saved in **local storage**, so data persists even after refreshing the page
- **Edit and delete** meals anytime
- Switch between **Light and Dark mode** for a comfortable viewing experience

---

- HTML
- CSS
